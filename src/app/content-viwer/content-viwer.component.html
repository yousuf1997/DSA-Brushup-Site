Array
Data Structure
What is Array?

An Array is a group of elments that are identified by an index. First index of the array will be 0, and the last index will be n-1 (n : number of elements). In memory the elements are stored next to each other.

Conceptual illusration of Array
Advantages of Array
  • Easy retrieval of an element, it will take only O(1) time.
  • 2D Array repersents matrices.
  • Easy to apply or implement sorting algorithms to sort the array.
  • It's convenient to repersent many data strucutures such as Trees, and Hash Map.
Disadvantages of Array
  • The size of an array is fixed.
  • To extend the array, you have to create a new array with greater size, and then copy the elments from old array to the new one
  • If the index of an element is unknown, then linear iteration has to be performed to check for an element.
Array in Java.

To delcare Array [We will declaring Array of int]

int items = new int[4];

To Add an element

items[0] = 13;
items[1] = 5;
items[2] = 45;
items[3] = 98;

Invalid Index

items[4] = 76;

An index 4 is invalid, even though we know that the array can have 4 elements, but the last index is always n - 1, in this case n = 4, then the last index we can have is 4 - 1 which is 3. Any thing greater than this index will results in Index Out Of Range Exception.

To iterate elements in Array

for(int i = 0; i < items.length; i++) {
   System.out.println(items[i]);
