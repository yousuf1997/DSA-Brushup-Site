What is HashMap?

HashMap is a collections which stores key and value as a pair. Retrieve values based on the key. It allows faster reterival of an element if the key is already known. The first parameter is the key and the second is the value. Both needs to be an object. Primitives are not allowed in the generic.

How to initialize a HashMap?

Map interface can be used to initialize or direct implementation of HashMap can be used as well.

Map<Integer, String> hashMap = new HashMap<>();

How to add elements to HashMap?

hashMap.put(1 , "one");
hashMap.put(2 , "two");
hashMap.put(3 , "three");
hashMap.put(4 , "four");

How to check if an element or key exists?

  • We check if a key exits by

    • hashMap.containsKey(3);
  • We check if an element exits by

    • hashMap.containsValue("three");
