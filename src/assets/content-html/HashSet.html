What is HashSet?

HashSet is a collections which stores unique values and no duplicates allowed. The insertions order is not maintained.

Conceptual illustration of HashSet :


How to initialize a HashSet?

Set interface can be used to initialize or direct implementation of HashSet can be used as well.

Set<Integer> hashset = new HashSet<>();

How to add elements to HashSet?

hashset.add(43);
hashset.add(56);
hashset.add(78);
hashset.add(90);

Time complexity : O(1)

How to check if an element is present in the HashSet?

hashset.contains(90);
Time complexity : O(1)

How to remove elements?

hashset.remove(90);
Time complexity : O(1)

How to iterate HashSet?

for(Integer val : hashSet) {
   System.out.println(val);
}
Time complexity : O(n)
