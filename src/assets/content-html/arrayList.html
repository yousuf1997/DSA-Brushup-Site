What is ArrayList

How to initilize ArrayList?

We are allowed to use only the wrapper classes for the collections generic type.

ArrayList<Integer> items = new ArrayList<Integer>();

How to add elements to the ArrayList?

items.add(34);
items.add(42);
items.add(11);
items.add(97);
items.add(87);

How to check if an element is present?

if presents then it will return true else false

items.contains(11);

How to get an element of an index?

Make sure the index is within the range otherwise it will throw IndexOutOfBoundException

items.get(1);
The above code will return 42

How to set an element to an index?

First paramenter is the index, and the second parameter is the element you will add it to that index.

items.set(1, 90);

How to remove an element by an index?

Make sure the index is within the range otherwise it will throw IndexOutOfBoundException

items.remove(1);
The above code will remove the value 42

How to check the size of the ArrayList?

items.size();

How to check if the ArrayList is empty?

items.isEmpty();
The above code will retrun true if the ArrayList has any elements, otherwise false.

How to iterate the ArrayList?

  1. for(int i = 0; i < items.size(); i++) {
       System.out.println(items[i]);
    }

  2. for(Integer i : items) {
       System.out.println(items[i]);
