What is Deque?

Behavior of a Deque is similar Queue, and with addition that the elements can be added in the front and removed from the front as well. It is combination of LIFO and FIFO.

How to initialize a Deque?

Deque itself is an interface, so we are going to use ArrayDeque implementation. We can also use LikedList implementation.

Deque<Integer> deque = new ArrayDeque<Integer>();

How to add elements to the Deque (Add a like regular queue, add from the back)?

deque.addLast(34);
deque.addLast(42);
deque.addLast(11);
deque.addLast(97);
deque.addLast(87);

How to add elements to the Deque (Add from the front)?

deque.addFirst(14);
deque.addFirst(62);
deque.addFirst(81);
deque.addFirst(70);
deque.addFirst(71);

How to check if an element is present in the Deque?

deque.contains(97);

How to get the front element without removing it?

deque.peekFirst();

How to get the last element without removing it?

deque.peekLast();

How to get the front element and remove it?

deque.pollFirst();

How to get the last element and remove it?

deque.pollLast();

How to iterate the Deque and remove the elements?

The following remove the elements like normal queue (LIFO), and you can also use pollLast() if you would like to remove from the last.

while(!deque.isEmpty()){
  System.out.println(deque.pollFirst());
}
