What is TreeMap?

This is map inserts elements based on their natural ordering. All the elements will be ordered.

How to initialize a TreeMap?

SortedMap interface can be used to initialize or direct implementation of TreeMap can be used as well.

TreeMap<Integer, String> treeMap = new TreeMap<>();

How to add elements to HashMap?

treeMap.put(3 , "Three");
treeMap.put(1 , "One");
treeMap.put(4 , "Four");
treeMap.put(2 , "Two");

Internally the elements will be ordered as ([1,"One"],[2,"Two"],[3,"Three"],[4,"Four"]).

Time complexity : O(log n)

How to check if an element or key exists?

  • We check if a key exits by

    • treeMap.containsKey(3);
  • We check if an element exits by

    • treeMap.containsValue("three");

Time complexity : O(log n)

How to replace an element in HashMap?

We need to have a key. The first parameter is the key, and the second parameter is the value.

treeMap.replace(3, "number three");

Time complexity : O(log n)

How to replace an element in HashMap?

We need to have a key, and the pass the key to the following function.

treeMap.remove(3);

Time complexity : O(log n)

How to get the key set?

The following will return Set<Integer>

treeMap.keySet();

How to get the values list?

The following will return Collection<Integer>.

treeMap.values();

How to iterate HashMap?

for(Map.Entry<Integer, String> entry : treeMap){
   System.out.println("Key : " + entry.getKey() " , Value : " + entry.getValue());
}

Time complexity : O(n)
