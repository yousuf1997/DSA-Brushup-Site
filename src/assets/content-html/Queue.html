What is Queue?

Behavior of a Queue is similar to the real life queue. In which the first person that entered the queue will be removed first. Likewise the elements are inserted and removed in First In First out (FIFO) manner.

How to initialize a Queue?

Queue itself is an interface, so we are going to use ArrayDeque implementation. We can also use LikedList implementation.

Queue<Integer> queue = new ArrayDeque<Integer>();

How to add elements to the Queue?

queue.add(34);
queue.add(42);
queue.add(11);
queue.add(97);
queue.add(87);
Time complexity : O(1)

How to check if an element is present in the Queue?

queue.contains(97);
Time complexity : O(n)

How to get the top element without removing it?

queue.peek();
Time complexity : O(1)

How to get the top element and remove it?

queue.pop();
Time complexity : O(1)

How to iterate the Queue and remove the elements?

while(!queue.isEmpty()){
  System.out.println(queue.poll());
}
Time complexity : O(n)
