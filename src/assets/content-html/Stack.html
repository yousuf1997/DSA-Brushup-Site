What is Stack?

Stack is data strucutre where elements are inserted and removed in Last In First Out manner. Think of it as stacking plates, and removing the plates from the top.

What is the base interface of Stack?

List (Interface) -> LinkedList (Class) -> Stack (Class)

How to initilize Stack?

We are allowed to use only the wrapper classes for the collections generic type.

Stack<Integer> stack = new Stack<Integer>();

How to add elements to Stack?

stack.push(34);
stack.push(42);
stack.push(11);
stack.push(97);
stack.push(87);

How to check if an element is present?

If presents it will return true else false.

stack.contains(97);

How to search for an element?

It will return the position of the element.

stack.search(97);

How to get the top element without removing it?

stack.peek();

How to get the top element and also removing it?

stack.pop();

How to iterate the stack and removing the elements?

while(!stack.isEmpty()){
  System.out.println(stack.pop());
}
