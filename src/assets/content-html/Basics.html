What is Array?

A collections of elements that are stored in contigious memory.

How to initialize Array?

You have to specify the number of elements when you initialize elements.

int[] array = new int[2];

How to add elements to Array?

You have to add elements by index, the first index will be always 0. So if the array has size of 2, then the first index 0, and last index is 1 (size of array - 1).

array[0] = 9;
array[1] = 4;

How to find the length of Array?

array.length;

How to iterate the Array?

for(int i = 0; i < array.length; i++){
   System.out.println(array[i]);
}

How to sort the Array?

Arrays.sort(array);

How to sort the Array with lambda (Above JDK 8)?

Arrays.sort(array, (a,b) -> { // logic });

How to copy Array?

Arrays.asCopy(array);

How to convert Array to List?

Arrays.asList(array);

What is Char?

A char is type which repersents a single character or symbol. Unlike String, the char takes only one symbole and enclosed with single quote.

How to intialize a Char?

char c = 'a';

How to check if a char is a digit?

Character.isDigit('1');

How to convert a char digit to a int?

Character.getNumericValue('1');

How to check if a char is space?

Character.isSpaceChar(' ');

What is String?

A String is basically array of characters. Usually contains text. String is immutable, which means programmer assigns value to a String, and reassign another value to the same variable, the previous variable will remain in the string pool.

How to initialzie a String?

  • There are two ways to initialize the string.
    1. Initialize a String using string literals.
      String s = "I am a String Literal";
    2. Initialize a String using String object.
      String s = new String("I am a String Object");

How to check if two strings are equal?

This is bit tricky becuase we may assume that we may '==' operator to compare two strings. This double equal operator compares the memory locations of a string by default. If you have two strings with exactly the same content, and one of the string was intialized using the literals, and other one was using the object. Depsite the content is the same but each of the content resides in different memory locations thus it returns false. The correct way to compare is follows

string1.equals(string2);
If you would like to ignore the case.

string1.equalsIgnoreCase(string2);

How to get a particular character of a String?

Since the String is array of characters, we can access the character by the index. Make sure the index you pass to the chatAt() does not exceed the size of the string n - 1.

string1.charAt(2);

How to check if particular sequence of a character exists in the String?

string1.contains("water");

How to get length of a String?

string1.length();

How to iterate the characters of a String?
