What is PriorityQueue?

A PriorityQueue is as name suggest that the elements will be organzied depends on the priority of the elmement. Internally it maintains heap, and there are two types of heap min and max. The top value of min heap wil always smaller than the rest of the values in the queue and max is vice versa. By default it orders the elment its according the natural ordering of the object usually this is done using compareTo() internally.

How to initialize PriorityQueue?

PriorityQueue<Integer> pq = new PriorityQueue<Integer>();

How to add elements to the PriorityQueue?

pq.add(34);
pq.add(42);
pq.add(11);
pq.add(97);
pq.add(87);
The elments above will be stored in sorted order (default : ascending) despite the insertion order. Sorted order : 11, 34, 42, 87, 97. As you can see by default its using min heap.

How to check if an element is present in the PriorityQueue?

pq.contains(97);

How to get the top element without removing it?

pq.peek();

How to get the top element and also remove it?

pq.pop();
